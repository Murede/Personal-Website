Transfer Learning on CIFAR

This project compares several transfer-learning paths to evaluate image classification performance on CIFAR datasets.

Overview

The notebook includes a from-scratch convolutional baseline, an ImageNet transfer path, a Places365 transfer path, and a multi-stage path that moves through CIFAR100 before the final CIFAR10 task. The aim was to determine when pretrained representations provide a measurable advantage over training from scratch.

Build

The implementation was structured as a reproducible experiment pipeline in PyTorch with shared data loading, controlled seeds, repeated runs, and aligned evaluation functions. The notebook reports Top-1 accuracy, macro-F1, learning curves, and confusion matrices.
