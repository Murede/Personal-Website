Function Generator Circuit Simulation

This project documents a function generator design modeled in Micro-Cap using a waveform source, active analog stages, and a passive conditioning network.

Overview

The simulation file represents a function-generator circuit intended to shape and manage an output waveform through analog components. The design references a triangle-wave source, op-amp models, and multiple resistor-capacitor elements used for timing and conditioning.

Build

The project was created as a Micro-Cap .cir schematic file with embedded model definitions and component metadata. Those references include the waveform model, amplifier stages, and passive network elements that define how the simulated signal is generated and shaped inside the circuit environment.
