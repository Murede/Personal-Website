BART Summarization Pipeline

This project tests whether continued self-supervised adaptation improves summarization performance for a pretrained BART model on XSum news data.

Overview

The notebook compares two starting points for abstractive summarization. One model begins from the original pretrained BART checkpoint, while the other begins from a checkpoint first adapted through masked denoising on article text.

Build

The workflow was organized in three stages: dataset loading, continued denoising training, and side-by-side fine-tuning under the same supervised setup. The final comparison uses ROUGE scores and generated summaries.
