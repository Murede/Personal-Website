Function Generator Circuit Simulation

This report summarizes the Micro-Cap circuit file used to model a function-generator design built from a waveform source, active amplifier stages, and a passive timing network.

Micro-Cap Analog Design Op-Amp Modeling Waveform Generation

What the project is about

The project focuses on modeling and validating a function-generator circuit in simulation. The file references a triangle-wave source, multiple resistor and capacitor elements, and op-amp models that shape or buffer the generated signal.

How it was created

The simulation is stored as a Micro-Cap .cir file with component definitions, waveform models, and schematic metadata. The attached visual is built from that file’s own references, including the triangle-wave model and op-amp model definitions, and presents the circuit as a readable high-level architecture.

Project artifact

The report image highlights the same core structure visible in the circuit file: waveform source, amplifier stages, and passive conditioning network. The download contains the original circuit files used for the simulation work.
