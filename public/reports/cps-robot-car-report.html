Cyber-Physical Robot Car Control

This article summarizes a sensor-driven robot-car control program that combines perception, state logic, and actuator output inside a single Python control loop.

Python Infrared Sensors Ultrasonic Avoidance Motor Control

What the project does

The program reads three infrared sensors to determine whether the car is centered on a line, drifting, at a junction, or off track. Based on that state, it issues motor commands for straight motion, turning, or recovery behavior.

How it was built

The implementation is organized around a Car class that initializes servo, ultrasonic, motor, infrared, ADC, buzzer, and LED modules. The main behavior runs in mode_infrared(), where sensor reads, line-following logic, and obstacle responses are all coordinated.
