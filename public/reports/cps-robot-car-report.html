Project Report

Cyber-Physical Robot Car Control

This project combines sensing, logic, and actuation in a Python control program for a small autonomous car.

Project Overview

The system reads infrared line-following sensors to determine vehicle position relative to a path and then updates motor behavior accordingly. It also integrates ultrasonic obstacle detection so the car can interrupt normal motion, stop, signal, reverse, and attempt a local recovery maneuver when the path ahead is blocked.

How It Was Created

The implementation is centered on a Car class that manages the hardware modules used in the lab setup, including the motor driver, infrared sensors, ultrasonic sensor, servo, LEDs, buzzer, and ADC. The control logic in mode_infrared() translates sensor states into directional decisions and then into motor commands.
