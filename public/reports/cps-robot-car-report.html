Cyber-Physical Robot Car Control

This short report summarizes the robot-car control program written for a cyber-physical systems lab, combining sensor input, decision logic, and actuator output in one control loop.

Python Infrared Sensors Ultrasonic Avoidance Motor Control

What the project is about

The program drives a small autonomous car by reading infrared line-following sensors, classifying the vehicle state, and issuing motor commands for straight motion, turning, junction handling, or recovery when the line is lost. It also adds obstacle avoidance using ultrasonic distance measurements.

How it was created

The implementation uses a Car class that initializes and manages multiple hardware modules: servo, ultrasonic sensor, motor driver, infrared sensors, ADC, buzzer, and LEDs. The report image is derived directly from the logic inside mode_infrared() and the obstacle-response branch that stops, signals, reverses, and turns when the path is blocked.

Project artifact

The visual in this report is not a generic diagram. It is a compact representation of the actual control path implemented in the source file, showing how sensor readings flow into decisions and then into motor and feedback outputs.
