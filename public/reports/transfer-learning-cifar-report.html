Project Report

Transfer Learning on CIFAR

This project investigates how different transfer-learning strategies affect image classification performance on CIFAR datasets. Instead of relying on a single baseline, the work compares several training paths so the results are easier to interpret and defend.

Project Overview

The notebook was built as a comparative study. It includes a from-scratch convolutional baseline, an ImageNet transfer-learning path, a Places365 transfer path, and a multi-stage transfer path that moves through CIFAR100 before the final CIFAR10 task. The main purpose was to determine when pretrained representations provide a measurable advantage over training from scratch.

How It Was Created

The implementation was structured as a reproducible experiment pipeline in PyTorch. Shared data-loading utilities, controlled seeds, repeated runs, and common evaluation functions were used so that the comparisons stayed aligned across all parts of the study. The notebook also reports Top-1 accuracy, macro-F1, learning curves, and confusion matrices so the analysis is not based on one metric alone.
