Transfer Learning on CIFAR

This short report summarizes the AISE3010 notebook that compares several image-classification training paths under a controlled experimental setup.

PyTorch Transfer Learning ResNet50 CIFAR10 / CIFAR100

What the project is about

The notebook investigates whether pretrained models improve classification performance compared with training from scratch. It covers a custom convolutional baseline, ImageNet transfer learning with two fine-tuning strategies, a Places365 transfer path, and a multi-stage path that adapts through CIFAR100 before the final CIFAR10 target.

How it was created

The project was built as a reproducible experiment pipeline with deterministic seeds, shared loaders, reusable training utilities, and reporting blocks for accuracy, macro-F1, learning curves, and confusion matrices. The visual above is derived from the notebook structure itself and mirrors the four-part evaluation flow implemented in the source file.

Project artifact

The report image is a compact rendering of the notebook’s experiment design: Part A scratch CNN, Part B ImageNet transfer, Part C Places365 transfer, and Part D multi-stage transfer. The downloadable notebook is available from the project card for full code and setup details.
